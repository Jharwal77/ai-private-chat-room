# AI Private Chat Room 💬🤖

A full-stack AI-powered private chat application built using React.js, Node.js, Express.js, and Gemini AI.

## Features ✨

* Smart AI-generated replies
* Session-based chat history
* Responsive modern chat UI
* REST API integration
* Real-time message rendering

## Tech Stack 🛠️

* React.js
* Node.js
* Express.js
* Gemini AI
* JavaScript

## Installation ⚙️

```bash id="2r2zk7"
git clone https://github.com/Jharwal77/ai-private-chat-room.git
cd ai-private-chat-room
npm install
```

## Run Project 🚀

